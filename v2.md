# V2 Development Plan - Organick E-commerce

## Overview
Fullstack E-commerce application with new features: WebSocket Chat, Coupon System, Product Reviews, Notifications, Shop UI Redesign, Admin Filters

---

## Changes Summary

### New Features Added

#### 1. WebSocket Chat (Realtime)
- **Backend**: Socket.io server, Chat model, Chat routes
- **Frontend**: Chat modal, Floating button, Admin chat page
- **Features**:
  - Room-based chat (user + admin)
  - Online/offline status
  - Chat history stored in MongoDB
  - Typing indicators

#### 2. Coupon System
- **Backend**: Coupon model, CRUD API, apply logic
- **Frontend**: Admin coupon management, Apply at checkout
- **Features**:
  - Percentage & fixed amount discounts
  - Min order requirement
  - Usage limit (1 per user)
  - Expiry date
  - Unique coupon codes

#### 3. Product Reviews
- **Backend**: Review model, CRUD API, purchase verification
- **Frontend**: Star rating, Review form, Review list
- **Features**:
  - 1-5 star rating
  - Comment text
  - Edit within 5 days of purchase
  - Only users who bought can review
  - One review per user per product

#### 4. Notifications
- **Backend**: Notification model, API, triggers
- **Frontend**: Bell icon, Notification panel
- **Features**:
  - In-app notifications only
  - Order status changes
  - Chat messages
  - Real-time via Socket.io
  - Mark read/delete

#### 5. Shop UI Redesign
- **Frontend**: New shop page layout
- **Design**: Based on organick-template.webflow.io
- **Features**:
  - Hero section
  - Filter sidebar (category, price, sort, search)
  - Product grid (3-4 columns)
  - New ProductCard with hover effects
  - Rating display
  - Sale badges

#### 6. Admin Table Filters
- **Frontend**: Reusable AdminTableFilters component
- **Apply to**: Products, Orders, Users, Categories, Coupons, Reviews tables
- **Features**:
  - Global search
  - Column filters (dropdown, date range)
  - Sort by column
  - Pagination

## Technical Implementation

### Backend Changes

#### New Models
1. **Coupon** (`BE/src/models/coupon.ts`)
   - code (unique string, uppercase)
   - type: 'percentage' | 'fixed'
   - value: number
   - minOrderAmount: number
   - maxUses: number (null = unlimited)
   - usedBy: [{ userId, usedAt }]
   - expiresAt: Date
   - isActive: boolean

2. **Review** (`BE/src/models/review.ts`)
   - userId, productId, orderId (refs)
   - rating: 1-5
   - comment: string
   - createdAt, updatedAt
   - editDeadline: Date (createdAt + 5 days)
   - isDeleted: boolean

3. **Notification** (`BE/src/models/notification.ts`)
   - userId, type: 'order' | 'chat' | 'review'
   - title, message, data (JSON)
   - isRead: boolean
   - createdAt

4. **Chat/Conversation** (`BE/src/models/conversation.ts`)
   - conversationId: string (userId + adminId)
   - participants: [userId, adminId]
   - messages: [{ senderId, content, timestamp, isRead, isDeleted }]

#### New Routes
- `POST /api/coupons` - Create coupon (admin)
- `GET /api/coupons` - List coupons (admin)
- `PUT /api/coupons/:id` - Update coupon (admin)
- `DELETE /api/coupons/:id` - Soft delete (admin)
- `POST /api/coupons/apply` - Apply coupon (user)

- `POST /api/reviews` - Create review (verify purchase)
- `PUT /api/reviews/:id` - Update review (within 5 days)
- `DELETE /api/reviews/:id` - Soft delete
- `GET /api/reviews/product/:productId` - List reviews

- `GET /api/notifications` - List user notifications
- `PUT /api/notifications/read/:id` - Mark as read
- `PUT /api/notifications/read-all` - Mark all read
- `DELETE /api/notifications/:id` - Delete

- `GET /api/chat/history/:userId` - Get chat history
- `DELETE /api/chat/message/:id` - Delete message

#### Socket.io Events
- `connection` / `disconnect`
- `join_room` / `leave_room`
- `send_message` / `receive_message`
- `typing` / `stop_typing`
- `user_online` / `user_offline`

### Frontend Changes

#### New Components
1. **Shop Components** (`FE/src/components/shop/`)
   - `CategoryFilter.tsx` - Multi-select chips
   - `PriceRangeFilter.tsx` - Min/Max slider
   - `SortFilter.tsx` - Dropdown
   - `SearchFilter.tsx` - Text input
   - `FilterSidebar.tsx` - Combine all filters
   - `ProductCardNew.tsx` - Redesigned card with hover

2. **Review Components** (`FE/src/components/review/`)
   - `StarRating.tsx` - Interactive/display rating
   - `ReviewForm.tsx` - Create/edit review
   - `ReviewList.tsx` - Paginated review list

3. **Chat Components** (`FE/src/components/chat/`)
   - `FloatingChatButton.tsx` - Bottom-right button
   - `ChatModal.tsx` - User chat interface
   - `AdminChatPage.tsx` - Admin chat dashboard

4. **Notification Components** (`FE/src/components/notification/`)
   - `NotificationBell.tsx` - Icon with badge
   - `NotificationPanel.tsx` - Dropdown list

5. **Admin Components** (`FE/src/components/admin/`)
   - `AdminTableFilters.tsx` - Reusable filter component

#### New Hooks
- `useSocket.ts` - Socket.io connection
- `useChat.ts` - Chat conversation
- `useNotifications.ts` - Notification management
- `useOnlineStatus.ts` - Online users tracking

#### New Pages
- `FE/src/app/(client)/shop/` - New shop layout
- `FE/src/app/(admin)/admin/chat/` - Admin chat page
- `FE/src/app/(admin)/admin/coupons/` - Coupon management
